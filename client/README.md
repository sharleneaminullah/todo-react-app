# ToDo App

## Installation and Setup Instructions

Clone down this repository. You will need `node` and `npm` installed globally on your machine.

Installation:

`npm install`

To Run Test Suite:

`npm test`

To Start Server:

`npm start`

To Start Server and Client together

`npm run dev`

To Visit App on desktop:

`http://localhost:3000/`
